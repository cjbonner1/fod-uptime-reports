# Glossary

## Acronyms
| Term | Meaning | Context |
|------|---------|---------|
| FoD | Fortify on Demand | OpenText AppSec platform |
| SLA | Service Level Agreement | FoD target: 99.9% uptime |
| SAST | Static Application Security Testing | Aviator service (eu/ams) |
| GMT | Greenwich Mean Time | All status page times are GMT |
| YTD | Year to Date | Calculated from completed months only |
| De-dup | De-duplication | Grouping simultaneous outages within 60s |

## Internal Terms
| Term | Meaning |
|------|---------|
| Bucket | Regional/service grouping (FedRAMP Portal, AMS API, etc.) |
| Vulncat | Vulnerability Categorization service |
| SAST Aviator | Static analysis service; eu-sast-aviator and ams-sast-aviator variants |
| status page | https://status.fortify.com/history |
| the uptime report | FoD_Uptime_Report_2026.xlsx |
| build script | Python script (openpyxl) that generates the workbook from scratch |
